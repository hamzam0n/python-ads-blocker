# python-ads-blocker
python ads blocker
	python3 adblock.py <option>
		option: 
			[+] block => To block ads
			[+] unblock => To unblock ads
		ex: 
				 sudo python3 adblock.py block
				 sudo python3 adblock.py unblock

 	PS: You need to run this code as *root* or *administrator* to run it successfully
